Release Notes

7.0.0

Release Date: 15 Aug 2024

  • FORGE-528 - Upgrade for XM 16, JDK 17 compatibility.

6.0.0

Release Date: 23 Aug 2022

  • HIPFORGE-421 - Upgrade for XM 15 compatibility.

5.0.0

Release Date: 12 May 2021

  • HIPFORGE-405 - Upgrade for XM 14 compatibility.
  • The artifacts' groupId was changed from org.onehippo.forge.doc-commenting to org.bloomreach.forge.doc-commenting (but no repackaging of Java classes though).
  • Added bootstrapping of domain /hippo:configuration/hippo:domains/doccommenting to the plugin, instead of doing this in an implementing project. Please check the upgrade path: best approach is to first remove this node from project yamls, then redo any changes.
